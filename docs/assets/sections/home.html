What is Wakanda ?

Wakanda is a full stack javascript platform for creating web, hybrid and native applications. Wakanda is composed of two main products:

  • Studio: IDE that helps you troubleshoot your environment, develop your apps and test them locally.
  • Server: hosts and runs your Wakanda backend application during development and production.

Wakanda Studio gives you a complete view of your application from the front to the back :

  • Application Scaffolding: compose your app by selecting from a preconfigured list of templates :
    • Backend: JavaScript, TypeScript..
    • Web: Angular4, Angular1.. (soon Aurelia and ReactJS)
    • Mobile: Ionic2, Ionic1.. (soon React Native)
  • environment troubleshooting: dev tools and system configuration problems detection and solutions are prepared for you, so that you can concentrate on creating value.
  • Frameworks CLI integration: generate framework elements for Angular and Ionic directly from the studio via a graphical user interface . Under the hood Wakanda studio makes use of angular-cli, ionic cli..
  • Code Versioning: Git integration.
  • Debugger: Debug your backend code directly from the studio.
  • Tasks: manage your app's running tasks (transpilation, filesystem watchers, livereloaders..), examine their output, stop, reload tasks.. from a centralized view for backend, mobile and web.
  • Code edition and completion: completion for JS/TS/HTML/CSS.. and data model aware code completion for JS/TS backend files.

Wakanda Server is a multi-threaded server that lowers the cost of configuration and developement by providing pre-bundled components :

  • JavaScript Application Server
    • Uses V8 as JavaScript runtime.
    • Exposes a rich synchronous API to handle filesystem, sockets, HTTP Requests, built-in or remote databases, workers..
    • Workers API so that you can use NodeJS modules, APIs and asynchronous style coding.
  • HTTP Server: exposes Wakanda's auto-generated REST API and can serve your application's static files.
  • Data Abstraction Layer
    • Handles the connection with data-sources (built-in WakandaDB or remote databases).
    • Exposes a rich JavaScript API to access and manipulate data.
    • Provides events associated to data manipulation.
    • Secures access to data.
    • Exposes an automatically generated REST API to access and manipulate data.
  • WakandaDB (built-in dataBase)
  • Authentication (Directory)
    • Handles users/groups and sessions management as well as authentication.
    • Exposes a Server Side JavaScript API.
    • Exposes a REST API.
