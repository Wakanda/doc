Prerequisites

All you need to follow along is a wakanda studio that can be downloaded from here.

Once the studio is installed, open it up and create your solution. In the solution creation dialog you can pick a name for your solution and select the project types and technologies you are interested in.

For this tutorial we will call our solution todo and we will be using the following projects :

  • Backend : Wakanda JavaScript
  • Web : Angular 4

Depending on the project types and technologies you've selected, dependencies and system configuration can be required, but don’t panic we’ve got your back. Once an issue is detected or a dependency is missing it will be displayed in the studio console.

For example, by clicking on Install Angular CLI you will receive a full description of what is needed to fix the issue.

After resolving all the issues detected by the studio troubleshooting, click the Run toolbar button. Behind the curtains an npm install followed by an npm start will be triggered.

Keep your application running and transpiling while we go through the tutorial.

  • You can select the project types you want to run.
  • If a project is already running it'll be stopped before getting run once again.
