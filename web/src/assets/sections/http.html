HTTP connection limitations


<settings>
	<http maxConnectionCountPerClient="50" 
		maxQueuedConnectionCount="512"
		maxQueuedConnectionCountPerClient="50"/>
</settings>
  • maxConnectionCountPerClient: limits the number of connections which are handled simultaneously for a client, Once the client reaches the limit, other connections are pushed in the queue.
  • maxQueuedConnectionCount:limit the size of the connections queue. Once the queue is full, the server responds with the error 503. maxQueuedRequestCount is global for all clients.
  • maxQueuedConnectionCountPerClient limit the number of queued connections per client this time.
