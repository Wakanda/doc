License

A license.key file is required to enable some features on Wakanda Enterprise Server. You can get one at Wakanda.io.

How to use your license.key file

Default locations

You can put your license file in one of the default locations:
  • Windows: C:/Users/<user>/Documents/Wakanda/license/license.key
  • Mac: /Users/<user>/Documents/Wakanda/license/license.key
  • Linux: /home/<user>/.wakanda/UserDoc/Wakanda/license/license.key
  • All platforms: SOLUTION/license/license.key

Custom location

You can specify your license file path when running wakanda Server
  • Windows:
    > wakanda-server.exe -l path/to/license.key
    > wakanda-server.exe --license path/to/license.key
  • Mac:
    $ wakanda-server -l path/to/license.key
    $ wakanda-server --license path/to/license.key
  • Linux:
    $ wakanda-server -l path/to/license.key
    $ wakanda-server --license path/to/license.key
